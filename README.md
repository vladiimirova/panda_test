# Weather App

Стартовый проект для приложения погоды на Vue 3 + Vite без CSS-фреймворков и UI-библиотек.

## Запуск

```bash
npm install
npm run dev
```

Для работы с OpenWeatherMap нужно создать `.env` по примеру `.env.example`.

## План по коммитам

1. Каркас проекта и стартовый интерфейс.
2. Подключение OpenWeatherMap API, определение города пользователя по IP, состояния загрузки и ошибок.
3. Блок погоды: автокомплит городов, текущая погода, переключатель день/5 дней.
4. График Chart.js без `vue-chartjs`.
5. Несколько блоков погоды, добавление до 5 штук, удаление через модальное подтверждение.
6. Вкладка избранного, localStorage, лимит 5 городов и модальные сообщения.
7. Мультиязычность en/uk, адаптивность 360-1200px и финальная полировка.
